# TelegramMC
Telegram Bot to manage minecraft servers

This is a Maven project :)

## commands:
* server <name> cmd <minecraft command>
  --> executes command in server <name>
* server <name> start
  --> starts server <name>
* server <name> stop
  --> stops server <name>
* server <name> save
  --> pushes server <name> to git
* server <name> load
  --> pulls server <name> from git
